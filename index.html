Máquina de Tag Post.

Este é um simulador de funcionamento da Máquina de Tag Post, desenvolvido para a disciplina de Teoria da Computação, do 3º ano de Bacharelado em Ciências da Computação, da FEMA (Fundação Educacional do Município de Assis). Desenvolvido utilizando Java 7, este software visa demonstrar como são realizadas as quatro operações básicas: adição, subtração, multiplicação e divisão.
